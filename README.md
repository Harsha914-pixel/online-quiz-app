# Online Quiz App

An interactive full-stack quiz application where users can take quizzes, view scores, and track their performance. Administrators can manage questions and quiz content through a dedicated admin panel.

## Features

### User Features
- User Registration & Login
- JWT Authentication
- Take Quizzes
- Instant Score Calculation
- View Results
- Leaderboard
- Responsive Design

### Admin Features
- Add Questions
- Update Questions
- Delete Questions
- Manage Quiz Content

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT (JSON Web Token)
